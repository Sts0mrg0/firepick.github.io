Affordable

FirepPick was designed to keep costs low, we are using cheap industry standards for the aluminium frame and the main kinematic is almost completely 3d printed.

Open Source

Using open licenses for both hardware and software allows the project to continually evolve in real time as developers add to it and modify it and you won't be locked in to a vendor.

Hackable

Access to all design files allows you to start building your own FirePick from scratch, fix issues as they occur and modify it to your personal prefernces.
